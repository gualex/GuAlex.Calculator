## GuAlex.Calculator

Калькулятор специального назначения :)

### Возможности

* Поддерживаются целые числа со знаком (Int32)
* Операция сложения +
* Операция вычитания -
* Операция умножения *
* Операция деления /
* Круглые скобки
* Унарные минус/плюс
* Обнаружение ошибок
  * арифметическое переполнение
  * деление на ноль
  * некорректная структура выражения
  * индикация места ошибки
